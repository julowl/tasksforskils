Python Функции — Краткая теория (не более 3 минут чтения)

Коротко о главном:

1. Что такое функция?
- Функция — именованный блок кода, который можно вызывать с аргументами и который может возвращать значение.
- Определяется через def имя(параметры):

2. Параметры и аргументы
- Позиционные параметры: def f(a, b)
- Параметры со значением по умолчанию: def f(a, b=2)
- *args — произвольное число позиционных аргументов (кортеж)
- **kwargs — произвольное число именованных аргументов (словарь)

3. Возврат значений
- return возвращает значение; функция может вернуть любое значение или None.

4. Функции как объекты
- Функции — объекты первого класса: их можно присваивать, передавать и возвращать.

5. Замыкания
- Замыкание — функция, которая «запоминает» переменные окружения, в котором была создана.
- Используется для создания параметризованных функций.

6. Рекурсия
- Функция может вызывать сама себя; нужна базовая точка останова, чтобы избежать бесконечного цикла.

7. Декораторы
- Декоратор — функция, принимающая функцию и возвращающая новую функцию.
- Применения: кэширование, логирование, повторные попытки и т.д.
