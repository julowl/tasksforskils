Циклы for в Python — краткая теория (≈ 3 минуты чтения)

- Назначение: цикл for итерирует элементы любого итерируемого объекта (списки, кортежи, строки, словари, файлы, генераторы).
- Базовый синтаксис:

  for item in iterable:
      # действие с item

- range: range(start, stop[, step]) генерирует целые числа. Частые шаблоны:
  - for i in range(n):  # 0..n-1
  - for i in range(a, b):  # a..b-1
  - for i in range(a, b, step)

- Полезные функции с циклами:
  - enumerate(iterable)  -> возвращает пары (индекс, элемент)
  - zip(*iterables)  -> пары/тройки элементов
  - reversed(seq)  -> итерирование в обратном порядке

- Управляющие операторы:
  - break: выйти из цикла раньше
  - continue: перейти к следующей итерации
  - else у цикла: выполняется, если цикл завершился без break

- Вложенные циклы: используются для работы с матрицами или декартовыми произведениями.

- Генераторы и итерация: можно использовать yield внутри цикла, чтобы лениво возвращать элементы.

Советы:
- Предпочитайте прямую итерацию по элементам, а не по индексам, если индекс не нужен.
- Используйте списковые/словарные включения (comprehensions), когда нужно создать коллекцию из цикла — это короче и чаще быстрее.
