Функции — теория (кратко)

1. Что такое функция?
- Функция — повторно используемый блок кода, который выполняет задачу и может быть вызван с аргументами для получения результата.

2. Подпись функции и аннотации типов
- Функция имеет имя, параметры, необязательные значения по умолчанию и необязательный возвращаемый тип. Используйте аннотации типов для документации: def f(x: int) -> int.

3. Позиционные, именованные, *args и **kwargs
- Позиционные аргументы сопоставляются по позиции. Именованные — по имени. *args собирает дополнительные позиционные аргументы в кортеж. **kwargs собирает дополнительные именованные аргументы в словарь.

4. Функции высшего порядка
- Функции могут принимать другие функции и возвращать функции. Примеры: map, filter, sorted с key.

5. Замыкания
- Замыкание — функция, которая захватывает переменные из внешней области видимости. Полезно для хранения состояния (например, счетчики, мемоизация).

6. Декораторы
- Декоратор — вызываемый объект, который преобразует функцию, часто для добавления поведения: логирования, кеширования, валидации. Используется с синтаксисом @decorator.

7. Рекурсия
- Функция, вызывающая сама себя. Подходит для деревьев и вложенных структур. Всегда обеспечивайте базовый случай.

8. Lambda и функциональные инструменты
- lambda создаёт небольшие анонимные функции. В сочетании с map, filter и reduce позволяют компактные преобразования данных.

Советы
- Предпочитайте читаемый код и документируйте требования в докстрингах. Используйте unit-тесты для проверки поведения.
