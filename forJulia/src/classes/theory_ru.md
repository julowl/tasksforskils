Классы в Python — Краткая теория (чтение до 3 минут)

1. Что такое класс?
- Класс — это шаблон (blueprint) для создания объектов (экземпляров). Класс объединяет данные (атрибуты) и поведение (методы).

2. Экземплярные и класс-атрибуты
- Экземплярные атрибуты хранятся в каждом объекте отдельно (self.x).
- Класс-атрибуты общие для всех экземпляров (MyClass.count).

3. Методы и параметр self
- Обычные методы экземпляра получают объект как первый аргумент (обычно self).
- Классовые методы получают класс как первый аргумент (обычно cls) и маркируются @classmethod.
- Статические методы не получают автоматически ни экземпляр, ни класс и маркируются @staticmethod.

4. Наследование и полиморфизм
- Класс может наследоваться от другого для повторного использования или расширения поведения. Подклассы могут переопределять методы и обращаться к реализации родителя через super().
- Полиморфизм позволяет работать с разными объектами через общий интерфейс.

5. Специальные (магические) методы
- Python поддерживает специальные методы вроде __init__, __repr__, __str__, __eq__, __add__, __len__ и т.д. Они позволяют объектам вести себя как встроенные типы (например, поддерживать оператор + или преобразование в строку).

6. Свойства и дескрипторы
- @property позволяет предоставлять вычисляемые атрибуты с методами-геттером, сеттером и делетером. Это даёт доступ в виде атрибутов, но позволяет выполнять валидацию или ленивые вычисления.

7. Композиция и фабричные методы
- Композиция значит строить классы, содержащие другие объекты (has-a), а не наследоваться (is-a).
- Фабричные методы (часто @classmethod) дают альтернативные конструкторы и удобные способы создания объектов из сторонних представлений (например, из словарей).

Зачем упражнения?
- Реализация классов помогает понять, где хранить состояние (класс против экземпляра), когда переопределять поведение и как сделать объекты похожими на встроенные типы. Упражнения фокусируются на этих нюансах.

Откройте файлы упражнений в src/. В каждом упражнении есть заглушки (raise NotImplemented) и тесты. Заполните заглушки и запустите тесты для проверки.
